What is the AI API Doctor Score Challenge?

This is a public testing activity for AI API users. You can use AI API Doctor to generate a diagnostic report, then share a one-line result in forums or group chats.

Your shared score helps:

  • Discover misjudgments and compatibility issues
  • Surface CORS issues and provider billing risks
  • Build community transparency around relay stations
Recommended Format

Post in forums, group chats, or comment sections with the following format:

My AI API Doctor Score: {score}/100 | API: {apiScore}/100 | Model: {modelScore}/100 | {mainFinding} | Report ID: {reportId} https://aiapidoctor.com/en/
If you don't want to run the Model Sanity Test, you can use "Model: Not tested".
Optional Supplementary Info

If you'd like, you can add the following (not required):

Interface: OpenAI Chat Encountered CORS: No What could be better: Nothing
This information helps others understand how different relay stations perform in different scenarios.
Security Reminder
Do NOT share the following:
  • Full API Key
  • Account password
  • Sensitive balance screenshots
  • Provider names you don't want to disclose

The diagnostic report only shows reproducible technical signals. It does not prove intentional overbilling by the provider, nor does it prove model authenticity. Model Sanity Score is only for detecting obvious anomalies or degradation — not an official model ranking.

How to Participate
  1. Open AI API Doctor Diagnostic Lab and fill in your Base URL and API Key
  2. Click "Start Diagnosis" and wait for the test to complete
  3. Click "Copy One-Line Result" and paste it into forums or group chats
  4. Optional: Enable "Run Model Sanity Test" for a more complete report

FAQ

Q: Do I need to install the Chrome extension?

No. The web version can complete all tests including the Model Sanity Test.

Q: Do I need to give a positive review or force-share?

No. You only need to post a one-line result. No extension installation, no review, no forced recommendation required.

Q: Will my provider know what I tested?

AI API Doctor does not upload results to any third party. Diagnostic requests are sent only to the Base URL you entered.

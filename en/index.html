Why Do You Need This Tool?

Many AI API relay stations use a pre-charge mechanism. Failed requests, empty replies, upstream 503 errors, timeouts, and cache hits can make it difficult for users to determine whether their actual billing is reasonable. AI API Doctor uses reproducible signals to help users and providers communicate.

Five Diagnostic Dimensions

Billing Integrity
Check whether failed requests and empty replies ultimately reduce quota. Wait 10 seconds to confirm pre-charge refunds.
Model Connectivity
Verify whether Base URL, API Key, model permissions, and HTTP status are functioning correctly.
Usage Integrity
Check whether the response returns prompt_tokens, completion_tokens, and total_tokens.
Cache Hit
Send two identical requests to check whether cached_tokens or cache_read_input_tokens are returned.
Price Audit
Calculate expected cost from usage and prices, compare with actual deduction to detect anomalies.

# AI API Doctor

API 配置风险检测工具 / API Configuration Risk Checker

**License: MIT**

AI API Doctor 是一个开源工具，帮助开发者检测 API 配置在兼容性、透明度、稳定性和客户端接入方面的风险。

## Features

- 核心 API 兼容性检测
- usage 扣费透明度审计
- 稳定性与延迟采样
- 模型身份验证
- 缓存命中信号检测
- 客户端配置完整性

提交问题时请附带这些信息

以下信息有助于快速定位问题。注意：只提供必要信息即可，不要上传完整 API Key。

  • AI API Doctor 版本号（插件设置页可见）
  • 浏览器版本（如 Chrome 126）
  • 操作系统（如 macOS 14、Windows 11）
  • Base URL 域名（如 api.example.com，不要提供完整 API Key）
  • 模型 ID（如 gpt-4o、claude-3-5-sonnet）
  • HTTP 状态（如 503、200）
  • 报告截图或复制的报告文本
  • 是否已登录 New API / One API 控制台
  • raw quota 是否可读
不要提交完整 API Key、账号密码、余额截图中的敏感信息。报告会自动脱敏 API Key（如 sk-****abcd），请确认报告文本中不包含完整 Key 再提交。

支持范围

AI API Doctor 可以帮助排查：
  • 失败请求是否扣 raw quota
  • 空回复是否扣费
  • 预扣是否在 10 秒内返还
  • Base URL / API Key / Model ID 是否可用
  • 报告如何发给站长或客服
AI API Doctor 不能保证：
  • 证明服务商故意多扣费
  • 100% 覆盖所有中转站模板
  • 代替法律意义上的审计报告
  • 读取所有非 New API / One API 站点的余额

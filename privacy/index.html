隐私说明

了解 AI API Doctor 如何处理你的数据。

一、AI API Doctor 处理哪些数据？

AI API Doctor 处理以下数据：

  • 用户填写的 Base URL
  • 用户填写的 API Key
  • 用户填写的模型 ID
  • New API / One API 原始额度（raw quota）
  • 余额变化记录
  • HTTP 状态码
  • 诊断报告内容

二、API Key 的存储方式

Chrome 插件中的 API Key 只保存在当前浏览器本地（chrome.storage.local），用于向你填写的 Base URL 发起诊断请求。

网页版手动报告生成器不要求输入 API Key，只接收 raw quota、HTTP 状态和模型名等公开信息。

三、API Key 的使用范围

API Key 仅用于向用户填写的 Base URL 发起诊断请求。

AI API Doctor 不会将 API Key 上传到 AI API Doctor 服务器。

四、报告自动脱敏

插件生成报告时只显示 API Key 前缀和尾号，不显示完整 Key。例如：sk-****abcd。

五、数据使用限制

AI API Doctor 承诺：

  • 不会把 API Key 上传到 AI API Doctor 服务器
  • 不会出售用户数据
  • 不会将用户数据用于广告画像
  • 不会将用户数据用于信用评估或贷款用途

六、Chrome Web Store Limited Use 声明

The use of information received from Chrome APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

七、安全边界

AI API Doctor 只展示本次测试中的可复现信号，不证明服务商故意多扣费。报告更适合用于排查、客服沟通和站长修复，不构成财务证明或法律依据。

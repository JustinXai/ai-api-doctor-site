真验中转站到底扣没扣钱

网页只能看返回结果和账单明细；插件可以在 New API / One API 控制台读取原始额度，检测请求前后额度变化。

Chrome 插件审核中
审核通过后这里会跳转到 Chrome Web Store
插件能测这些（网页验不了的）
空值到底扣没扣
发一个空值请求，对比请求前后的原始额度变化
失败请求到底扣没扣
发一个必定失败的请求，看额度有没有变化
返回废包到底扣没扣
HTTP 200 但实际没产物，看有没有扣钱
预扣有没有返还
观察预扣额在请求结束后是否正确返还
插件怎么工作的
1 插件读取 New API / One API 控制台的原始额度（raw quota）
2 自动发一个测试请求（空值、失败、或正常请求）
3 请求结束后再次读取原始额度
4 对比前后变化，生成真实验货报告
隐私说明
API Key 保存在浏览器本地，不会上传到任何第三方服务器。插件只在本地读取控制台额度数据，所有计算都在你的浏览器内完成。

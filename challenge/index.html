AI API 体检分挑战

生成你的 AI API 体检报告，贴出一行结果即可参与。不需要公开 API Key，也不需要公开服务商名称。

什么是 AI API 体检分挑战？

这是一个面向 AI API 用户的公开测试活动。你可以用 AI API Doctor 生成一份体检报告，然后在论坛或群聊中贴出一行结果。

你的晒分可以帮助：

  • 帮助其他用户发现误判、兼容性问题
  • 暴露 CORS 问题和服务商扣费风险
  • 建立社区对各中转站透明度的共识
推荐格式

在论坛、群聊或评论区贴出以下格式（一行或多行均可）：

我的 AI API 体检分：{score}/100｜API：{apiScore}/100｜模型：{modelScore}/100｜{mainFinding}｜报告 ID：{reportId} https://aiapidoctor.com/
如果不想开启模型表现检测，模型部分可以写"模型：未检测"。
可选补充信息

如果你愿意，可以补充以下信息（不强制）：

接口类型：OpenAI Chat 是否遇到 CORS：否 哪里不好用：无
这些信息可以帮助其他人了解不同中转站在不同场景下的表现。
安全提醒
不要贴以下内容：
  • 完整 API Key
  • 账号密码
  • 敏感余额截图
  • 你不想公开的站点名称

体检分报告只展示技术信号，不证明服务商故意多扣费，也不代表模型真假。模型表现分仅用于发现明显降智或能力异常，不代表官方模型排名。

常见问题

Q：我需要安装插件吗？

不需要。网页版即可完成全部检测，包括模型表现分。

Q：需要给好评或强制分享吗？

不需要。你只需要贴一行体检分，不需要安装插件，不需要给好评，不需要强制推荐。

Q：服务商知道我测了什么吗？

AI API Doctor 不会把结果上传到第三方。检测请求只发送到你填写的 Base URL。

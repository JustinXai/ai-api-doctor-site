为什么需要这个工具？

很多 AI API 中转站采用预扣费机制。问题是：失败请求、空回复、上游 503、超时或无有效产物请求，有时也可能留下最终扣费。AI API Doctor 用 raw quota 证据链帮助你判断：这次请求到底有没有最终扣余额。

三个核心卖点

读取 raw quota，不看前台余额
前台余额常常只显示两位小数，一次 $0.0001 级别的变化看不出来。raw quota 可以看到更细粒度的额度变化。
等待 10 秒确认最终结算
部分站点会先预扣再返还。必须等待结算完成后，才能区分"临时预扣"和"最终扣费"。
生成可发给站长的报告
报告包含 HTTP 状态、模型名、raw quota 变化、时间、脱敏说明和站长修复建议，便于客服沟通。
